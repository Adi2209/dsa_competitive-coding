## Description 
Element can be removed from First using pop method, remember it will remove the first element inserted only.

## Method
```cpp
queue<int> first;
// Adding Element for demo
first.push(1);
first.push(2);
first.push(4);
first.push(6);

// popping (removing) elements 
first.pop(); // 1 removed
first.pop(); // 2 removed
```

### Queue now contains
```cpp
4 6
```
