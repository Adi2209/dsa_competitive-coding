## Question Link :

https://codeforces.com/problemset/problem/677/A

## Code solution :

```cpp

#include<bits/stdc++.h>
using namespace std;
int main(){
    // Input
    int n,h;
    cin>>n>>h;
    int a[n],width=0;
    // Looping and if height is greater than h, adding 2 else 1 and printing
    for(int i=0;i<n;i++){
        cin>>a[i];
        if(a[i]>h)width+=2;
        else width+=1;
    }
    cout<<width<<endl;
return 0;
}

```
