## Question Link :

https://codeforces.com/problemset/problem/486/A

## Code solution :

```cpp

#include<bits/stdc++.h>
using namespace std;
int main(){
    // Input
    long long int n;
    cin>>n;
    // Short trick, as 2 number will add to '1'
    if(n%2==0)cout<<n/2<<endl;
    else cout<<n/2-n<<endl;
return 0;
}

```
