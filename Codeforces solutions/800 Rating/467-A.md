## Question Link :

https://codeforces.com/problemset/problem/467/A

## Code solution :

```cpp

#include<bits/stdc++.h>
using namespace std;
int main(){
    // Input
    int n,output=0;
    cin>>n;
    while(n--){
        // Input of each room and adding 1 if room has atleast 2 capacity free and printing
        int a,b;
        cin>>a>>b;
        if(b-a>=2)output++;
    }
    cout<<output;
return 0;
}

```
